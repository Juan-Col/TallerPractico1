Guía Definitiva: HTML

Seleccione un ejercicio para practicar las etiquetas solicitadas en clase.

Ejercicios

• Ejercicio #1
H1-H6, varios <p>, <strong>, <em>. Tema: HTML como lenguaje de marcado (no programación).
• Ejercicio #2
H1, H2, strong, em, <blockquote> y ejemplo de bloque de código con <pre> <code>.
• Ejercicio #3
<ins>, <del>, <abbr>, <acronym>, <dfn>, <dl>, <dt>, <dd>, y <blockquote>. Incluye etiquetas personalizadas INCS y DELS.

Recuerde: todas las páginas deben incluir las meta etiquetas charset, description, keywords y author.
